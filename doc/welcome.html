Welcome to Sparklehoof

A node-based image- and video-processing playground.
Offline welcome page

Getting started

  1. Create a flow by typing a name and pressing Create, or open an existing .flowjs file.
  2. Drop nodes onto the canvas from the node list on the left.
  3. Wire them up by dragging from an output port to an input port.
  4. Press Run and watch the result update in the viewer panel.

Node categories

Sources

Load images and video from disk, patterns, or generators.

Filters

Dither, threshold, scale, split and recombine channels.

Sinks

Write the final frames out to files or the viewer.

Tips

  • Your most recent flows appear on the start page for quick access.
  • User-defined nodes live under ~/.image-inquest/user_nodes/.
  • Run with --flow NAME to jump straight into a saved flow.
