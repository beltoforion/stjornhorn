Welcome to Stjörnhorn v0.1.19

A node-based image- and video-processing playground.

Getting started

  1. Create a flow — type a name and press Create, or open an existing .flowjs.
  2. Drop nodes from the list on the left onto the canvas.
  3. Wire them up by dragging output ports to input ports.
  4. Press Run and watch the viewer panel update.

Node categories

Sources

Load still images (JPEG, PNG, CR2) or video files (MP4, AVI, MOV, MKV) as streams of frames.

Filters

Scale, shift, blur, threshold, dither, split/join RGBA channels, overlay with per-pixel alpha, and match templates with NCC.

Sinks

Write frames to image files or encode video (MP4V, XVID) with live preview via the Display node.

What's new in v0.1.19

  • Value Source & Value Sink — a numeric counter source that emits a SCALAR per frame (parameters: min_value, max_value, multiplier, loop) and a matching sink that terminates a numeric flow. Drop them on the canvas with a Display in between to watch the counter live in the inline preview.
  • Display now shows SCALAR & MATRIX — feed a scalar stream into Display and the inline preview renders the formatted number; matrices render as a compact text grid. Image payloads keep their pixmap preview.
  • SCALAR & MATRIX payload types (v0.1.18) — internal groundwork for value-driven nodes; existing flows are unaffected.
  • Backdrops (v0.1.17) — right-click empty canvas to drop a coloured frame behind a group of nodes. Use them as loose chapter headings ("Colour prep", "Alpha mask") so large flows stay readable.
  • Video Source paths are now portable (v0.1.16) — video references under the input/ folder save as short relative names and resolve them the same way as image sources, regardless of the working directory.
  • Unsaved-changes indicator (v0.1.15) — the editor's toolbar shows an amber warning icon with "Unsaved changes" the moment a parameter, node, or connection is edited, and clears it on save, load, or reload.
  • Reload toolbar action (v0.1.15) re-reads the current flow from disk, discarding unsaved edits after a confirmation.

Tips

  • Recent flows appear on the start page for quick access.
  • User-defined nodes live under ~/.image-inquest/user_nodes/.
  • Run with --flow NAME to jump straight into a saved flow.
  • Press F11 with the output inspector focused to toggle fullscreen preview.
